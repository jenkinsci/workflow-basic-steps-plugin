Conditionally executes a block of code. If called from within a stage and the condition is not met, the stage will be tagged as skipped. Example: 

stage('stage') {
    when(shouldSkip()) {
        sh 'process'
    }
}

The condition argument is designed to mimic the rules of Groovy Truth when possible:

  • Non-empty Collections and Arrays are coerced to true.
  • Non-empty Maps are coerced to true.
  • Non-empty Strings are coerced to true.
  • Non-zero Numbers are coerced to true.
  • Non-null object references are coerced to true.
